# 🌸 Iris Flower Classification using Machine Learning

## 📌 Project Overview

This project is a **Machine Learning web application** that classifies Iris flowers into three species:

* Setosa
* Versicolor
* Virginica

The classification is based on four input features:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

A trained ML model is integrated with a **Flask web application** to provide real-time predictions.

---

## 🚀 Features

* 📊 Predict Iris flower species instantly
* 🎯 Uses trained Machine Learning model
* 🌐 Interactive web interface
* ⚡ Fast API-based prediction system
* 🧠 Demonstrates core ML concepts (classification, scaling, encoding)
